# 画宗科技 — 制片资产管理平台 PRD

> **Product Requirements Document v1.0**
> **项目代号**: HuaZong Studio (画宗制片中枢)
> **基座项目**: CGWire Kitsu (Vue 3 + Flask + PostgreSQL)
> **目标**: 将开源制片管理系统改造为画宗科技内部全链路制片资产管理平台
> **部署目标**: 腾讯云（CVM + COS + PostgreSQL + Redis）
> **并发目标**: 50+ 人同时在线协作

---

## 一、产品愿景

将 Kitsu 从"动画/VFX 制片跟踪工具"改造为**画宗科技专属的全链路制片资产管理平台**，覆盖：

```
项目创建 → 分镜规划 → 资产上传 → 风格锁定 → AI 元数据提取 → 瀑布流审阅 → 标注协作 → 永久资产库 → 多项目复用
```

**核心差异点**（Kitsu 没有，我们必须新增）：
1. **AI 风格锁定引擎** — Gemini API 提取图片风格关键词并翻译为中文
2. **瀑布流分镜面板** — 导演视角的直观分镜浏览与标注
3. **永久资产库** — 跨项目的提示词/风格/镜头语言/人物/道具资产管理
4. **智能节点系统** — 每个资产自动生成关联节点图谱

## 二、用户角色

| 角色 | 权限 | 典型操作 |
|------|------|---------|
| **管理员** | 全部权限 | 创建项目、管理成员、系统配置 |
| **导演/总监** | 项目级全权 | 审阅分镜、标注反馈、风格锁定、资产审批 |

| **美术/动画师** | 任务级操作 | 上传分镜/资产、查看标注、下载参考 |
| **TD/技术** | 系统级配置 | API 集成、存储管理、自动化配置 |

### 1.2 部署架构搭建（腾讯云）

**目标**: 搭建腾讯云生产环境，支撑 50+ 并发用户

**架构设计**:
```
                    ┌─────────────────────────────────┐
                    │         腾讯云 CLB              │
                    │     (负载均衡 / HTTPS)           │
                    └───────────┬─────────────────────┘
                                │
                    ┌───────────┴─────────────────────┐
                    │                                  │
              ┌─────┴─────┐                    ┌──────┴──────┐
              │  CVM × 2  │                    │  CVM × 1    │
              │ Flask API  │                    │ Vue 前端     │
              │ (Gunicorn) │                    │ (Nginx)     │
              └─────┬─────┘                    └─────────────┘
                    │
        ┌───────────┼───────────────┐
        │           │               │
  ┌─────┴─────┐ ┌──┴───┐  ┌───────┴────────┐
  │ PostgreSQL │ │Redis │  │ 腾讯云 COS      │
  │ (云数据库)  │ │(缓存) │  │ (对象存储)      │
  └───────────┘ └──────┘  │ 图片/视频/文件   │
                          └────────────────┘
```

**关键配置**:
| 组件 | 规格 | 用途 |
|------|------|------|
| CVM (API) | 4C8G × 2 | Flask API + Gunicorn |
| CVM (前端) | 2C4G × 1 | Nginx 静态托管 |
| PostgreSQL | 2C4G 100GB | 核心数据库 |
| Redis | 2G | 会话缓存 + 消息队列 |
| COS | 按量 | 文件存储（图片/视频/资产） |
| CLB | 标准型 | HTTPS 负载均衡 |
| SSL | 免费证书 | HTTPS 加密 |

**智能体分工**:
| Agent | 职责 |
|-------|------|
| Agent-A（运维） | Docker Compose 编排、Nginx 配置 |
| Agent-B（后端） | Gunicorn 配置、Flask 生产模式 |
| Agent-C（数据库） | PostgreSQL 初始化、迁移脚本 |

**验收标准**:
- [ ] Docker Compose 一键启动全套服务
- [ ] HTTPS 访问正常
