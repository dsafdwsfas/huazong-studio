# 画宗制片中枢 — Worker 基础指令

你是画宗科技制片资产管理平台开发团队的一员。

## 项目背景
- 基于开源项目 Kitsu (CGWire) 改造
- 前端: Vue 3 + Vite + Vuex + Vue Router + Bulma CSS
- 后端: Flask + SQLAlchemy + PostgreSQL
- 目标: 改造为画宗科技内部全链路制片资产管理平台

## 代码库位置
- 前端: 当前工作目录 (kitsu-frontend)
- 后端 API: ../kitsu-zou/

## 开发规范
1. **保留 Kitsu 核心架构** — 不重写，渐进式改造
2. **遵循现有代码风格** — Vue 3 Options API (Kitsu 现有风格), Flask Blueprint 模式
3. **每个改动必须可测试** — 写完代码立刻验证
4. **中文注释** — 新增代码用中文注释
5. **不破坏现有功能** — 改造前先读懂现有代码

## 多智能体协同规则
你必须使用 Agent Teams 或多个子代理来并行完成你的任务:
- 将你的任务拆解为 2-4 个独立子任务
- 每个子任务派一个子代理执行
- 子代理之间不能有文件编辑冲突
- 所有子代理完成后，你负责集成验证

## 完成标准
- [ ] 代码无语法错误
- [ ] ESLint 无 error (warning 可接受)
- [ ] 新增功能有基本测试
- [ ] 不破坏现有功能
- [ ] 变更文件清单明确
